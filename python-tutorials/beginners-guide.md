# Python Beginner's Guide: Getting Started

Python is one of the most popular programming languages in the world, known for its simplicity and versatility. This guide will walk you through everything you need to know to get started with Python.

## Why Learn Python?

- **Easy to learn**: Simple syntax, reads like English
- **Versatile**: Web development, data science, automation, AI
- **Huge community**: Millions of developers, endless resources
- **Free**: Open source, no licensing fees
- **Cross-platform**: Works on Windows, Mac, Linux

## Setting Up Python

### Installation

1. Visit [python.org](https://python.org)
2. Download Python 3.x (latest version)
3. Run the installer (check "Add Python to PATH")
4. Verify installation: `python --version`

### First Python Script

Create a file called `hello.py`:

```python
print("Hello, World!")
```

Run it:
```bash
python hello.py
```

## Python Basics

### Variables and Data Types

```python
# Strings
name = "Martin"
greeting = f"Hello, {name}!"

# Numbers
age = 30
price = 19.99

# Booleans
is_adult = True

# Lists
fruits = ["apple", "banana", "cherry"]

# Dictionaries
person = {
    "name": "Martin",
    "age": 30,
    "city": "Hobart"
}
```

### Control Flow

```python
# If statements
age = 18
if age >= 18:
    print("You're an adult")
else:
    print("You're a minor")

# Loops
for fruit in fruits:
    print(fruit)

# While loops
count = 0
while count < 5:
    print(count)
    count += 1
```

### Functions

```python
def greet(name):
    return f"Hello, {name}!"

result = greet("Martin")
print(result)
```
